# DashOfDelish

Recipe app that allows users to create recipes and store with ease.  


## Installation

### Fork and clone repo
```
$ git clone https://github.com/yousifhj/DashOfDelish.git
```

### Switch to directory
```
$ cd DasOfDelish/
```

### Execute for dependencies
```
$ bundle install
```

### Seed database
```
$ rake db:migrate
```

### Startup the server
```
$ rails s
```

### Enter in browser to view
```
$  localhost:3000
```
